A comprehensive Python development practices guide covering Python selection and management, development tools (package management, formatting, and code checking), language syntax usage, testing tools, packaging tools, logging, virtual environments, and more. It's pretty much all-encompassing.

2„ÄÅ[State of the Art Python in 2024](https://4zm.org/2024/10/28/state-of-the-art-python-in-2024.html)

What are the best practices for creating Python projects in 2024? What tools are recommended? The author's list includes uv, ruff, pyproject.toml, pyright, pydantic, pytest, and click. The article received lots of upvotes and replies on Reddit.

3„ÄÅ[A Year In Review: Flask in 2024](https://blog.miguelgrinberg.com/post/a-year-in-review-flask-in-2024)

What interesting things happened in the Flask community in 2024? The author reviews Flask's development from several perspectives, including version releases, framework development speed, new features, backward compatibility, competition, and popularity.

4„ÄÅ[Introducing Annotated Logger: A Python package to aid in adding metadata to logs](https://github.blog/developer-skills/programming-languages-and-frameworks/introducing-annotated-logger-a-python-package-to-aid-in-adding-metadata-to-logs/)

GitHub's vulnerability management team has open-sourced a new logging library called Annotated Logger. It uses decorators to log function and class execution, supports custom log objects, sends JSON-formatted logs, and supports adding fields like Git branch and vulnerability names to logs.

5„ÄÅ[Cowboy Debugging Python Memory Leaks](https://rtpg.co/2025/01/01/cowboy-coding-memory)

How do you debug memory leak issues in Python? The author shares the `objgraph` library, which can view memory object growth and trigger runtime checks through signals. The article shows how to use it to locate and visualize memory leaks, helping to quickly identify and resolve memory leak problems.

6„ÄÅ[All your Python project in one file with shiv](https://www.bitecode.dev/p/all-your-python-project-in-one-file)

How do you use `shiv` to package a Python project and its dependencies into an executable zip file (zipapp)? How does shiv improve runtime speed? What other features does shiv support, and what issues should you watch out for when using it?

7„ÄÅ[Project Quarantine](https://blog.pypi.org/posts/2024-12-30-quarantine/)

The official PyPI blog introduces the project quarantine feature, which allows administrators to flag potentially harmful projects and prevent malicious software from being installed. The article covers implementation details of this feature. Since August 2024, about 140 projects have been flagged for quarantine, with only one being corrected.

8„ÄÅ[Debugging Python with VS Code and Sentry](https://blog.sentry.io/debugging-python-with-vs-code-and-sentry/)

How do you set up and use debugging features in VS Code? The article covers installing extensions, configuring debug environments, using breakpoints, adding variable monitoring, handling common exceptions like `FileNotFoundError` and `JSONDecodeError`, and shows how to use Sentry for continuous monitoring and debugging in production environments.

9„ÄÅ[Build your Jupyter dashboard using Solara](https://blog.jupyter.org/build-your-jupyter-dashboard-using-solara-569fa5b16e9c)

Introduces using Solara's UI components to create an interactive dashboard in Jupyter, with the final product allowing users to filter, visualize, and explore datasets on a map. The second part of the series covers how to deploy the service, and the third part introduces adding more components to extend the dashboard.

10„ÄÅ[Disposable environments for ad-hoc analyses](https://ericmjl.github.io/blog/2024/11/8/disposable-environments-for-ad-hoc-analyses/)

juv is a tool for simplifying Python environment management in Jupyter notebooks, allowing dependencies to be embedded directly in notebook files for easier setup and sharing. The article also covers integrating juv with pyds-cli for temporary data analysis, enhancing reusability and reducing environment conflicts.

11„ÄÅ[Python: spy for changes with sys.monitoring](https://adamj.eu/tech/2024/12/30/python-spy-changes-sys-monitoring/)

The `sys.monitoring` introduced in Python 3.12 can listen for specific events on particular code lines, providing fine-grained control. The article demonstrates through practical examples how to use it to monitor changes to sys.path, how to set up monitoring callback functions, and shows its powerful utility in debugging complex code.

12„ÄÅ[Implementation and Extension: Applications of the Boids Algorithm](https://people.ece.cornell.edu/land/courses/ece4760/labs/s2021/Boids/Boids.html)

The Boids algorithm is an artificial life program that simulates flocking behavior in birds, producing realistic natural bird dynamics. The article provides an in-depth explanation of Boids algorithm implementation, introduces how to implement Boids animation on TFT displays, and explores possibilities for adding complex behaviors like predators and roosting behavior.
