2„ÄÅ[Python Concurrency: Threads, Processes, and asyncio Explained](https://newvick.com/python-concurrency/)

3„ÄÅ[Why is hash(-1) == hash(-2) in Python? ](https://omairmajid.com/posts/2021-07-16-why-is-hash-in-python/)

What's going on? In Python, hash(-1) actually equals hash(-2), which is -2? The author dug through Python's official documentation and source code to track down the specific implementation of the `hash` function and finally found the reason: C language uses -1 to indicate error return values, so the hash() function might return -1 on error. If it correctly returns -1 as well, it would be confusing, so they agreed to use -2 as a replacement...

5„ÄÅ[PEP 769 ‚Äì Add a 'default' keyword argument to 'attrgetter' and 'itemgetter' ](https://peps.python.org/pep-0769/)

A new proposal suggesting adding a `default` keyword argument to the `attrgetter` and `itemgetter` functions in the `operator` module, so they can return default values when target attributes or items are missing, avoiding exceptions and simplifying code.

6„ÄÅ[Python Packaging: Why we can't have nice things, Part 2](https://zahlman.github.io/posts/2025/01/07/python-packaging-2/)

The article introduces its uses and some limitations, but mainly shares three tips for improving pipx efficiency: managing environments through "injecting" dependencies, using pipx's built-in pip for installations, and installing pipx itself through pipx to get updates.

7„ÄÅ[Django vs. FastAPI, An Honest Comparison](https://www.david-dahan.com/blog/comparing-fastapi-and-django)

8„ÄÅ[Weak references and garbage collectors](https://bernsteinbear.com/blog/weakrefs/)

The author once participated in developing a Python interpreter called Skybison, which used a moving garbage collector that organizes memory by moving objects and updating pointers. The article explains how to handle weak reference objects during garbage collection and how to implement this in custom garbage collectors.

9„ÄÅ[AI-text-to-video-model-from-scratch: build a small scale text-to-video model from scratch](https://github.com/FareedKhan-dev/AI-text-to-video-model-from-scratch)

10„ÄÅ[Python for DevOps](https://www.pulumi.com/blog/python-for-devops/)

11„ÄÅ[Building an Anemia Detection System Using Machine Learning](https://dev.to/yogesh10/building-an-anemia-detection-system-using-machine-learning-368i)

Anemia affects millions of people worldwide, and traditional detection methods are time-consuming, so the author developed a system that can quickly assess and accurately predict anemia. The project uses scikit-learn to implement a random forest classifier, develops a web interface based on Flask, and the code is open source.
