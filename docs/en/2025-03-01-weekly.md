Hey there! I'm Python Cat. Here I share quality Python, AI, and general tech content weekly, mostly in English. The newsletter is open source on [Github](https://github.com/chinesehuazhou/python-weekly) - if you like it, please give us a star! ⭐

This week we're sharing 12 articles and 12 open source projects.

## [🦄 Articles & Tutorials](https://xiaobot.net/p/python_weekly)

1、[A peek into a possible future of Python in the browser](https://lukasz.langa.pl/f37aa97a-9ea3-4aeb-b6a0-9daeea5a7505)

Explores the future of Python in browsers, introducing high-performance Python programming implemented by SPy. SPy, as an emerging "Python-like" language, achieves efficient code compilation and execution by dividing code into "blue code" executed at compile time and "red code" executed at runtime.

2、[Over the clouds: CPython, Pyodide and SPy](http://antocuni.eu/2025/02/26/over-the-clouds-cpython-pyodide-and-spy/)

A week-long collaboration/adventure experience between the SPy author and two other community leaders, sharing several problems they solved together, SPy's basic concepts and design decisions, and building a simple SPy playground. Particularly emphasizes the value of interpersonal relationships and the joy of spending time together.

3、[FastAPI Deconstructed: Anatomy of a Modern ASGI Framework](https://rafiqul.dev/blog/fastapi-deconstructed-anatomy-of-modern-asgi-framework)

Provides an in-depth analysis of FastAPI's architecture and core components, explaining how FastAPI works collaboratively through the ASGI protocol, Uvicorn server, Starlette framework, and Pydantic data validation library, showcasing features like dependency injection and automatic documentation generation.

4、[An Introduction to Flask-SQLAlchemy in Python](https://blog.appsignal.com/2025/02/26/an-introduction-to-flask-sqlalchemy-in-python.html)

Flask-SQLAlchemy enables seamless integration of SQLAlchemy with Flask. The article provides detailed coverage of its key features and demonstrates how to easily manage databases with Flask-SQLAlchemy through practical projects.

5、[The Best Way to Use Text Embeddings Portably is With Parquet and Polars](https://minimaxir.com/2025/02/embeddings-parquet)

How to efficiently use and store text embedding vectors? The article proposes a solution using Parquet file format and Polars library, enabling fast loading of embedding vectors and similarity calculations while supporting dynamic filtering and metadata management.

6、[To Type or Not to Type? ](https://blog.jonathanchun.com/2025/02/16/to-type-or-not-to-type/)

What value do Python's type hints provide? Besides adding code complexity, can they bring enough benefits? The article demonstrates the advantages of type hints from multiple perspectives, introduces techniques for effectively using type hints, and shows how to enhance development experience with modern tools.

7、[Which Python GUI library should you use in 2025?](https://www.pythonguis.com/faq/which-python-gui-library/)

The Python ecosystem has quite a few GUI libraries - which one is best suited for your project? The article compares multiple mainstream GUI libraries (Tkinter, PyQt, PySide, Kivy, PySimpleGUI, WxPython, PyGObject, Remi, etc.), introducing their relative advantages and licensing restrictions, and creates simple "Hello World" projects with each library.

8、[Building AI Agents for JupyterLab using Notebook Intelligence](https://blog.jupyter.org/building-ai-agents-for-jupyterlab-using-notebook-intelligence-0515d4c41a61)

Notebook Intelligence is an AI coding assistant and extensible AI framework that combines JupyterLab with AI. The article introduces how to define multiple tools (such as geographic coordinate queries, map generation, notebook creation and sharing) to build an AI agent.

9、[Run LLMs on macOS using llm-mlx and Apple's MLX framework](https://simonwillison.net/2025/Feb/15/llm-mlx/)

The article introduces how to run local large models on macOS, utilizing the llm-mlx plugin and Apple's MLX framework, demonstrating plugin installation, model downloading, and text generation with large models, along with model optimization and caching techniques.

10、[Introduction to CUDA Programming for Python Developers](https://www.pyspur.dev/blog/introduction_cuda_programming)

CUDA is a platform and C++ extension for writing programs that run on GPUs. The article explores how CUDA kernels work and how they map computations to GPU hardware.

![](https://img.pythoncat.top/2025-03-01-cuda.png)

11、[Design of Everyday APIs](https://roguelynn.com/words/everyday-apis)

How do you design an API with excellent user experience? The article proposes several fundamental principles and demonstrates how to optimize user experience through naming conventions, providing symmetric methods, default values, reducing repetition, and leveraging language features, using a publish/subscribe service API design as an example.

12、[Ugly Code and Dumb Things](https://lucumr.pocoo.org/2025/2/20/ugly-code/)

How do you choose and balance between pursuing code perfection and rapid delivery? Through personal experience and analysis of Flickr's Flamework framework, the author reveals that sometimes rapid iteration and solving real problems are often more important than code elegance.

## [🐿️ Projects & Resources](https://xiaobot.net/p/python_weekly)

1、[tinycompiler: Writing a compiler in a week-end](https://github.com/ssloy/tinycompiler)

A weekend project that supports translating the author's invented "wend" programming language into GNU assembly.

2、[django-rusty-templates: Django template language implemented in Rust](https://github.com/LilyFoote/django-rusty-templates)

100% compatible rendering output, error reporting as useful as Django's errors, with performance superior to pure Python implementations.

3、[when: Convert datetimes between timezones](https://github.com/dakrauth/when)

Converts time from one location to other places/timezones, can also display current moon phases, show full moon dates, and calculate past or future times.

4、[opendrop: An open Apple AirDrop implementation written in Python](https://github.com/seemoo-lab/opendrop)

A command-line tool for sharing files between devices via Wi-Fi, unique in its compatibility with Apple AirDrop, supporting file sharing with Apple devices running iOS and macOS. (8.9K stars)

5、[MakeItPrivate: Convert images to PDF](https://github.com/LittleYellowPanda/MakeItPrivate)

Uses FastAPI as backend and Streamlit as frontend, supports converting images (JPG, PNG) to PDF and vice versa, also supports combining multiple images and PDFs into a single PDF.

6、[codegen-sdk: a python library for manipulating codebases](https://github.com/codegen-sh/codegen-sdk)

Write code that transforms code. Combines Tree-sitter's parsing capabilities with rustworkx's graph algorithms to achieve scriptable, large-scale, multi-language code manipulation.

7、[mindmap-generator: Turn any input document into a sophisticated, context-dependent mindmap](https://github.com/Dicklesworthstone/mindmap-generator)

Utilizes large models to extract core concepts, relationships, and details from different types of documents and organize them into intuitive, hierarchical mind maps.

8、[probly: AI-powered spreadsheet application](https://github.com/PragmaticMachineLearning/probly)

An AI application that combines spreadsheet functionality with Python data analysis capabilities, supporting formulas, running Python code in browsers, data visualization, AI smart suggestions, and automated analysis.

![](https://img.pythoncat.top/2025-02-28-probly.png)

9、[gpustack: Manage GPU clusters for running AI models](https://github.com/gpustack/gpustack)

Open source GPU cluster manager with broad compatibility across various hardware and models, easy scaling of GPUs or nodes, distributed inference, multiple inference backends, user and API key management, GPU performance and utilization monitoring, token usage and rate analysis. (1.8K stars)

10、[PocketFlow: Minimalist LLM Framework in 100 Lines. Enable LLMs to Program Themselves.](https://github.com/The-Pocket/PocketFlow)

A minimalist LLM framework implemented in 100 lines of code, supporting multi-agents, workflows, and RAG.

11、[smolagents: a barebones library for agents. Agents write python code to call tools and orchestrate other agents.](https://github.com/huggingface/smolagents)

Open sourced by Hugging Face, run powerful agents with just a few lines of code, supports any LLM, supports text, visual, and even audio input. (12.8K stars)

12、[awesome-ai-agents: Awesome list of 300+ agentic AI resources](https://github.com/slavakurilyak/awesome-ai-agents)

AI agents outperform traditional LLMs - this is an awesome list of AI agent resources.
