# Python Trending Weekly #92: A Map of Python Packages

Hey there! I'm Python Cat. Here I share quality Python, AI, and general tech content weekly, mostly in English. The newsletter is open source on [Github](https://github.com/chinesehuazhou/python-weekly) - if you like it, please give us a star! ⭐

This week we're sharing 12 articles and 12 open source projects.

## [🦄 Articles & Tutorials](https://xiaobot.net/p/python_weekly)

1、[A Map of Python Packages](https://fi-le.net/pypi/)

PyPI hosts over 500,000 open source projects. The author used its JSON API and BigQuery dataset to generate an interactive package dependency graph, discovering some fascinating patterns.

![](https://img.pythoncat.top/2025-03-08-python-map.png)

2、[Exploiting Python pickles](https://davidhamann.de/2020/04/05/exploiting-python-pickle/)

What security issues exist with Python's `pickle` module? The article explains how pickle works, demonstrates how to exploit its deserialization functionality for remote code execution (RCE), and reminds us about secure handling methods when deserializing data from untrusted sources.

3、[Deep Dive into Django Signals: Structure, Use Cases, and Best Practices](https://awstip.com/deep-dive-into-django-signals-structure-use-cases-and-best-practices-ccbe1d3d5931)

Django signals are a powerful mechanism for decoupling application logic from event handling. The article explores Django signals' structure and working principles, introduces their use cases, and shares several best practices.

4、[Python 3.13 in 2025 Breakthroughs: No-GIL, JIT, and iOS Support Explained](https://www.techiediaries.com/news/python-313-2025-breakthroughs-no-gil-jit-ios-support-explained/)

The article summarizes major breakthroughs in Python 3.13: performance improvements (5-15% faster than 3.12), experimental JIT compiler, 7% reduction in memory usage, data science optimizations, iOS support, and no-GIL mode.

5、[PEP 775 – Make zlib required to build CPython](https://peps.python.org/pep-0775/)

This PEP proposes that starting from Python 3.14, building CPython will no longer support cases without the zlib compression library. The `zlib` module will become a required part of the standard library, with the only exception being the WASI platform.

6、[Python is not Java or C++: Python as a second language empathy](https://ballingt.com/python-second-language-empathy/)

What misconceptions might developers who use Python as a second language have? The author compares differences between Python and Java/C++, lists multiple confusing points and examples, and explores how to help developers with other language backgrounds better understand and use Python.

7、[Build a To-do App With Python and Kivy](https://www.pythonguis.com/examples/kivy-to-do-app/)

How do you develop a to-do application with Python? The article demonstrates building a graphical user interface to-do app using Kivy + SQLite, which can serve as a Python GUI beginner project.

![](https://img.pythoncat.top/2025-03-07-kivy.png)

8、[Visualizing process trees with marimo and anywidget](https://blog.cauchy.org/blog/anywidget/)

The article introduces how to develop an interactive process tree visualization tool using **marimo** and **anywidget**, building tree structures based on process creation event logs and rendering visualizations through **DependenTree**.

9、[Embedding Python in Elixir, it's Fine](https://dashbit.co/blog/running-python-in-elixir-its-fine)

Introduces how to embed a Python interpreter in Elixir, implementing automatic data conversion, code execution, and virtual environment management between Elixir and Python through Pythonx.

10、[Satellogic's Open Satellite Feed](https://tech.marksblogg.com/satellogic-open-data-feed.html)

Introduces Satellogic's open satellite data source "Satellogic EarthView" and demonstrates how to use various tools (such as GDAL, Python, DuckDB, and QGIS) to analyze and visualize this data.

11、[Django and Postgres for the Busy Rails Developer](https://andyatkinson.com/django-python-postgres-busy-rails-developer)

This is a practical guide for Rails developers transitioning to Django, comparing similarities and differences between Django and Ruby on Rails in development, including language features and runtime management, dependency management, migration mechanisms, Postgres configuration, code formatting, and linting.

12、[Debugging a Vision Transformer Compilation Issue](https://ohadravid.github.io/posts/2025-01-debugging-vit-and-tensorrt/)

The author encountered compilation issues when converting a Vision Transformer (ViT) model from PyTorch to TensorRT. The article demonstrates how to systematically debug and optimize deep learning models, such as using binary search to gradually isolate problems and comparing performance of different implementation approaches.

## [🐿️ Projects & Resources](https://xiaobot.net/p/python_weekly)

1、[Pokemon-Terminal: Pokemon terminal themes.](https://github.com/LazoVelko/Pokemon-Terminal)

719 unique Pokémon themes that can be set as desktop wallpapers and terminal backgrounds, supporting terminals including iTerm2, ConEmu, Terminology, Windows Terminal, Tilix, and Kitty. (4.6K stars)

2、[smallpond: A lightweight data processing framework built on DuckDB and 3FS](https://github.com/deepseek-ai/smallpond)

DeepSeek's open source data processing framework based on DuckDB, capable of scalably processing petabyte-level datasets. (3.9K stars)

3、[PRevent: Prevent merging of malicious code in pull requests](https://github.com/apiiro/PRevent)

A self-hosted GitHub app that listens to PR events, scans for malicious code, and comments detection results directly on PRs.

4、[stealthkit: Python requests on steroids](https://github.com/theonlyanil/stealthkit)

Simulates real user behavior by rotating user agents, setting referrers, handling cookies, managing proxies, and implementing retry logic.

5、[fastrtc: The python library for real-time communication](https://github.com/freddyaboulton/fastrtc)

Turns any Python function into real-time audio/video streams supporting WebRTC or WebSockets. (2.6K stars)

6、[VisualCrypto: A collection of Visual Cryptography and Random Grid schemes](https://github.com/coduri/VisualCrypto)

A toolkit specifically for Visual Secret Sharing (VSS), supporting Visual Cryptography (VC) and Random Grid (RG) technologies.

7、[awesome-stream-processing: A collection of demos showcasing how stream processing can be used to solve real-world problems](https://github.com/risingwavelabs/awesome-stream-processing)

Collects a series of executable demos showcasing stream processing applications in real-world scenarios.

8、[onyx: Gen-AI Chat for Teams - Think ChatGPT if it had access to your team's unique knowledge.](https://github.com/onyx-dot-app/onyx)

An AI platform that connects your company documents, applications, and team members. Feature-rich chat interface supporting knowledge sync and access control for over 40 applications. (12.2K stars)

9、[FoleyCrafter: AI Foley Master - Adding Sound Effects to Silent Videos](https://github.com/open-mmlab/FoleyCrafter)

A video-to-audio generation framework that can generate realistic sound effects semantically related to and synchronized with videos.

10、[SmartestKid: Windows Desktop AI Assistant](https://github.com/Alkali-Sim/SmartestKid)

Uses Windows COM automation to interface with Microsoft Office (Word, Excel), images, and file systems, suitable for exploring AI-driven desktop automation.

11、[Merlion: Merlion: A Machine Learning Framework for Time Series Intelligence](https://github.com/salesforce/Merlion)

End-to-end machine learning framework including loading and transforming data, building and training models, post-processing model outputs, and evaluating model performance. Supports various time series learning tasks including forecasting, anomaly detection, and change point detection for both univariate and multivariate time series. (4.2K stars)

12、[developer-portfolios: A list of developer portfolios for your inspiration](https://github.com/emmabostian/developer-portfolios)

This repository collects many developer portfolio homepages, providing product inspiration from others' homepages and works. (11.9K stars)
