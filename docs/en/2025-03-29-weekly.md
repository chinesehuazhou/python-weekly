This week we're sharing 11 articles and 12 open source projects.

## [🦄 Articles & Tutorials](https://xiaobot.net/p/python_weekly)

1、[How to Use Async Agnostic Decorators in Python](https://www.patreon.com/posts/how-to-use-async-124658443)

What challenges arise when you want to use decorators on both synchronous and asynchronous functions? The article introduces how to create decorators that work with both sync and async functions using generators. This article is from Patreon's tech blog.

2、[Serverless Python Websites](https://johnscolaro.xyz/blog/serverless-python-websites)

The author shares the process of completely refactoring a small Python project to serverless architecture, moving from VPS to AWS Lambda, reducing costs from $350 per year to $2 per month.

3、[A Simple Introduction to Common Load Balancing Algorithms](https://www.manjusaka.blog/posts/2025/03/23/a-simple-introduction-about-load-balance-algorithm/)

The article introduces implementations of common load balancing algorithms with Python code examples, including random selection, round-robin, least connections, and consistent hashing algorithms.

4、[An Unreachable Hidden XKCD Easter Egg inside CPython](https://blog.codingconfessions.com/p/cpython-hidden-xkcd-easter-egg)

Python has a famous xkcd comic easter egg: [import antigravity](https://xkcd.com/353/). Beyond that, the author accidentally discovered another xkcd easter egg about "unreachable state": when the impossible happens, anything becomes possible...

5、[Friends don't let friends write production software in Python](https://dustri.org/b/friends-dont-let-friends-write-production-software-in-python.html)

An author with eight years of Python experience vents about two major pain points: types and exceptions.

6、[Why we chose LangGraph to build our coding agent](https://www.qodo.ai/blog/why-we-chose-langgraph-to-build-our-coding-agent/)

The article shares Qodo's reasons and process for choosing the LangGraph framework to develop their AI coding assistant, introducing LangGraph's state machine model, clean API design, node-based architecture, and built-in state management features.

7、[A Python Test-Runner that generates correct implementations](https://claudio.uk/posts/unvibe-a-python-test-runner-that-generates-correct-implementations.html)

Introduces a Python tool called Unvibe that can generate correct code implementations through unit tests. Its principle is using unit tests as reward functions, calling large models through Monte Carlo tree search algorithms to generate code until all tests pass.

8、[Prospero challenge, now with more garbage collection](https://bernsteinbear.com/blog/prospero/)

Shares how to optimize a Python program through garbage collection, reducing runtime from 40 seconds to 10 seconds and memory usage from nearly 60 GB to 1 GB. Additionally, using CuPy instead of NumPy reduced runtime to 1.5 seconds.

9、[PyTorch internals](https://blog.ezyang.com/2019/05/pytorch-internals/)

A very long article that deeply introduces PyTorch's internal mechanisms, including tensor data structures, automatic differentiation mechanisms, memory management, and operation implementations.

10、[Programmer Reading List: 100 Tech Articles I Love (41-50)](https://www.piglei.com/articles/programmer-reading-list-3/)

The third part of the tech article sharing series by the author of "Python Craftsman", focusing on programmer fundamentals, software engineering, backend development, technical writing, Python, and Go languages.

11、[Reflections on Ten Years of Programming](https://ramsayleung.github.io/zh/post/2024/%E7%BC%96%E7%A8%8B%E5%8D%81%E5%B9%B4%E7%9A%84%E6%84%9F%E6%82%9F/)

The author shares years of thinking about programming technology and workplace life experiences. "A programmer's competitiveness isn't writing code, nor any particular language or framework - the core competitiveness is the ability to solve problems through technology."

## [🐿️ Projects & Resources](https://xiaobot.net/p/python_weekly)

1、[Agent-S: Agent S: an open agentic framework that uses computers like a human](https://github.com/simular-ai/Agent-S)

A modular and scalable computer-use framework designed to achieve autonomous interaction with computers and automatically execute complex tasks. (1.4K stars)

![](https://img.pythoncat.top/2025-03-28-agent.png)

2、[basedpyright: pyright fork with various type checking improvements](https://github.com/detachhead/basedpyright)

A fork of pyright with type checking improvements, enhanced vscode support, and pylance features built into the language server. (1.7K stars)

3、[gurubase: RAG system for creating AI Q&A assistants](https://github.com/Gurubase/gurubase)

Open source RAG system supporting multiple data sources including web pages, PDFs, videos, and GitHub repositories, for quickly developing website widgets, Slack bots, and Discord bots.

4、[compress_json: read and write compressed JSONs](https://github.com/LucaCappelletti94/compress_json)

Manipulate compressed JSON files, supporting gzip, bz2, and lzma (or xz) formats.

5、[lihil: Unbelievably fast async webframework](https://github.com/raceychan/lihil)

Pure Python web framework with features including dependency injection, OpenAPI docs & error response generator, problem pages, data validation & parameter parsing, built-in messaging system, and powerful AI support.

6、[fastapi_mcp: A zero-configuration tool for automatically exposing FastAPI endpoints as Model Context Protocol (MCP) tools](https://github.com/tadata-org/fastapi_mcp)

Automatically discovers all FastAPI endpoints and converts them to MCP tools, directly mounting MCP servers to FastAPI applications.

7、[arrg: building modular command-line applications](https://github.com/terror/arrg)

Develop modular command-line applications using a declarative, class-based approach.

8、[mcp-obsidian: MCP server that interacts with Obsidian](https://github.com/MarkusPfundstein/mcp-obsidian)

Enables MCP interaction with Obsidian through Obsidian's REST API community plugin, including listing directories and files, reading file content, searching files, and editing notes.

9、[local-deep-researcher: Fully local web research and report writing assistant](https://github.com/langchain-ai/local-deep-researcher)

Utilizes LLMs from Ollama or LMStudio to search the web, summarize, and reflect, ultimately generating Markdown summaries. (6.7K stars)

![](https://img.pythoncat.top/2025-03-28-deep-research.png)

10、[douyin-downloader: Douyin Batch Download Tool](https://github.com/jiji262/douyin-downloader)

Batch download Douyin content with watermark removal, supporting videos, image sets, collections, and music (original audio). Supports command-line parameters or YAML configuration file operation. (2.3K stars)

11、[fastopenapi: FastOpenAPI is a library for generating and integrating OpenAPI schemas](https://github.com/mr-fatalyst/fastopenapi)

Generate and integrate OpenAPI documentation using Pydantic v2 with various frameworks (Falcon, Flask, Quart, Sanic, Starlette, Tornado).

12、[star-vector: StarVector is a foundation model for SVG generation that transforms vectorization into a code generation task](https://github.com/joanrod/star-vector)

A multimodal vision-language model for Scalable Vector Graphics (SVG) generation, capable of performing image-to-SVG and text-to-SVG generation. (2.7K stars)
