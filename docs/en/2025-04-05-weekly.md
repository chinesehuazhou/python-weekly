## [🦄 Articles & Tutorials](https://xiaobot.net/p/python_weekly)

1、[A Deep Dive into MCP: The Revolutionary Breakthrough for AI Model Interaction with the External World](https://juejin.cn/post/7488895565314949120)

What exactly is the recently trending MCP? This article introduces MCP's technical background, advantages, and development process, showcasing its fundamental differences from traditional Function Calling technology, and demonstrates code examples for MCP client and server development. (Also: [What is MCP (Model Context Protocol)?](https://arthurchiao.art/blog/but-what-is-mcp/))

![](https://img.pythoncat.top/2025-04-05-mcp.png)

2、[We hacked Google's A.I Gemini and leaked its source code](https://www.landh.tech/blog/20250327-we-hacked-gemini-source-code)

A hacker team shares their experience discovering Gemini vulnerabilities during Google's LLM bugSWAT event and earning the "Most Valuable Hacker" title, detailing their approach to exploring Python sandbox environments.

3、[How We Migrated to Async Python](https://blog.duolingo.com/async-python-migration)

Duolingo's official blog shares their experience transforming a synchronous microservice into an asynchronous one, including how to reach consensus on migration, design async versions of sync libraries, and handle async code.

4、[Share Python Scripts Like a Pro: uv and PEP 723 for Easy Deployment](https://thisdavej.com/share-python-scripts-like-a-pro-uv-and-pep-723-for-easy-deployment/)

PEP-723 "Inline Script Metadata" specification allows embedding external dependencies directly into single-file scripts without needing additional `requirements.txt`. Combined with uv and shebang, it simplifies Python script distribution and execution.

5、[Building a Real-time Dashboard with FastAPI and Svelte](https://testdriven.io/blog/fastapi-svelte/)

Using Server-Sent Events (SSE) to push real-time data updates from FastAPI to a Svelte frontend, developing an interactive dashboard with real-time updates.

6、[Data Visualization Powerhouse — The Interesting Philosophy of Streamlit](https://www.qtmuniao.com/2025/03/18/streamlit/)

Streamlit is a Python library for rapidly developing simple web pages. The article analyzes its three fundamental design philosophies: writing frontend with backend language, rebuilding on new events, and supporting session-level caching.

7、[Optimizing Python: Understanding Generator Mechanics, Expressions, And Efficiency](https://pybit.es/articles/generator-mechanics-expressions-and-efficiency/)

Explores Python generator mechanics and demonstrates their advantages in memory efficiency and data processing through data comparisons.

8、[Unveiling 8 Hardcore Uses of Python Network Programming](https://juejin.cn/post/7488599657125396480)

Beyond chat rooms, what else can Socket do? The article introduces port scanners, simple HTTP servers, LAN games, remote control, file transfer, load balancing, DDOS attack simulators, and network proxy servers.

9、[PyTorch and Python Free-Threading](https://trent.me/articles/pytorch-and-python-free-threading/)

Introduces Python 3.13's "free-threading" mode and how it achieves efficient parallel model inference with PyTorch. By training GPT-2 models without GIL, it demonstrates the benefits of fully utilizing multi-core CPU resources in AI inference scenarios.

10、[How to Automate PyPI Publishing with uv, Trusted Publisher, and GitHub](https://blog.zech.codes/automate-uv-with-trusted-publisher)

Using uv to simplify Python package management, PyPI's Trusted Publisher for secure connections, and GitHub Actions to automatically trigger package uploads to PyPI.

11、[How I Automated My Podcast Transcript Production With Local AI](https://den.dev/blog/how-i-automated-podcast-transcription-with-local-ai/)

The author leverages WhisperX's speech-to-text and alignment capabilities to quickly process audio files, then uses local LLM to organize transcribed text and generate high-quality podcast subtitles.

12、[Opening a web page from Python](https://www.mostlypython.com/opening-a-web-page-from-python/)

The author discovered that using subprocess + shlex + open to open web pages has OS compatibility issues, while the standard library webbrowser easily solves this, so they analyzed webbrowser's implementation principles.

## [🐿️ Projects & Resources](https://xiaobot.net/p/python_weekly)

1、[plain: A web framework for building products with Python](https://github.com/dropseed/plain)

A web framework forked from Django, consisting of a core package and several optional third-party packages.

![](https://img.pythoncat.top/2025-04-04-plain.png)

2、[playa: PDF Parallel and Lazy Analyzer](https://github.com/dhdaines/playa)

Provides parallelizable lazy-loading interfaces to access PDF file internal structures; view various PDF metadata from command line; extract text or text objects, supports JPEG image extraction.

3、[safe-result: A Python package for elegant error handling](https://github.com/overflowy/safe-result)

Inspired by Rust's Result type, enables more explicit error handling without relying on try/catch blocks.

4、[clypi: Your all-in-one for beautiful, prod-ready CLIs](https://github.com/danimelchor/clypi)

Lightweight CLI framework that's type-safe, async-capable, easy to test, with composable commands and flexible configuration.

5、[gcop: AI-Powered Git Commit Assistant](https://github.com/undertone0809/gcop)

Transforms messy commits into meaningful narratives. Automates commit message generation, enhances Git workflow, and makes version control effortless with 20+ smart commands. (Shared by @Zeeland)

6、[conftier: Multi-layer Configuration Management Framework](https://github.com/Undertone0809/conftier)

Multi-layer configuration management framework similar to VSCode's user/workspace settings model, simplifying the definition, access, and synchronization of hierarchical configurations in Python programs. (Shared by @Zeeland)

7、[arxiv-mcp-server: A Model Context Protocol server for searching and analyzing arXiv papers](https://github.com/blazickjp/arxiv-mcp-server)

Filter arXiv papers by date and category, download and read paper content, view downloaded papers, store papers locally.

8、[nova-act: Develop agents that execute in browsers](https://github.com/aws/nova-act)

Amazon's open-source experimental SDK for developing agents that reliably execute actions in browsers.

9、[dockerpyze: Python applications to Docker, automatically.](https://github.com/nicoloboschi/dockerpyze)

Automatically generates Docker images for uv/poetry projects, PEP-621 compatible, supports adding configuration in pyproject.toml.

10、[E-book: Architecture Patterns with Python](https://www.cosmicpython.com)

Published by O'Reilly, with free online reading version. Helps Python developers manage architectural design for complex projects, including dependency inversion, domain-driven design, event-driven architecture, and reactive microservices.

11、[social-auto-upload: Automatically upload videos to Douyin/Xiaohongshu/WeChat Channels/Bilibili etc.](https://github.com/dreammis/social-auto-upload)

Automates video publishing to major social media platforms, supports scheduled uploads. (4.4K stars)

12、[VideoX-Fun: Generate videos of any resolution, supports image-to-video](https://github.com/aigc-apps/VideoX-Fun)

A video generation pipeline for creating AI images and videos, training Diffusion Transformer baseline models and Lora models.
