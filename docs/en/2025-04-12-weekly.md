## [🦄 Articles & Tutorials](https://xiaobot.net/p/python_weekly)

1、[NVIDIA Finally Adds Native Python Support to CUDA](https://thenewstack.io/nvidia-finally-adds-native-python-support-to-cuda/)

NVIDIA has implemented native Python support in the CUDA toolkit, elevating Python to first-class citizen status. Developers no longer need to understand C/C++ to use the CUDA toolkit more conveniently.

2、[Django: what's new in 5.2](https://adamj.eu/tech/2025/04/07/django-whats-new-5.2/)

Django 5.2 has been released! The article details multiple new features and improvements: automatic model imports, composite primary key support, form customization, content negotiation, template block tags, URL generation improvements, testing optimizations, and more.

3、[Owning My Own Data: Self-hosted Calendar Solutions](https://emilygorcenski.com/post/owning-my-own-data-part-1-integrating-a-self-hosted-calendar-solution/)

The author wants to break free from dependence on services like Google Calendar by taking control of personal calendar data through self-hosted solutions. The article covers the exploration process from CalDAV to iCalendar, using Python to implement calendar data format conversion.

4、[Build a Desktop Sticky Notes Application with PySide6 & SQLAlchemy](https://www.pythonguis.com/examples/pyside6-desktop-sticky-notes)

This tutorial provides detailed instructions on developing a desktop sticky notes application using PySide6 and SQLAlchemy, implementing borderless draggable notes, multi-note management, custom note styling, and system tray integration.

5、[PEP 784 – Adding Zstandard to the standard library](https://peps.python.org/pep-0784/)

A draft PEP proposing to add a new standard library with Meta's zstd library as the default implementation. Zstandard provides high compression ratios and fast decompression, superior to zlib and bzip2.

6、[Running Background Tasks from Django Admin with Celery](https://testdriven.io/blog/django-admin-celery)

How to handle long-running tasks in Django projects? How to integrate Celery with Django? How to trigger tasks from Django admin and get real-time task status?

7、[Algorithms for high performance terminal apps](https://textual.textualize.io/blog/2024/12/12/algorithms-for-high-performance-terminal-apps/)

The Textual project author shares technical details on implementing high-performance graphical user interfaces in terminals, including solving multi-window overlap issues, quickly determining which windows are in visible areas, and optimizing terminal app rendering speed.

8、[Reproducing word2vec with JAX](https://eli.thegreenplace.net/2025/reproducing-word2vec-with-jax)

word2vec is a model for generating word embeddings. The article details how to reimplement it using JAX.

9、[Improving Recommendation Systems & Search in the Age of LLMs](https://eugeneyan.com/writing/recsys-llm/)

How have recommendation systems and search technologies evolved in today's LLM era? The article discusses industry developments in search and recommendation systems over the past year, covering model architectures, data generation, training paradigms, and unified frameworks.

10、[Inside the CodeBot: A Gentle Introduction to How LLMs Understand Nullability](https://dmodel.ai/nullability-gentle)

Can large models correctly understand null values? This is crucial for code generation. The article introduces how to evaluate models' understanding of null values through model outputs and internal states, showing the impact of model training and code context on large models.

11、[Python's Most Banned Scripts: Getting You Blacklisted?](https://dev.to/snappytuts/pythons-most-banned-scripts-getting-you-blacklisted-55n4)

Why are some Python libraries blocked by cloud service providers? Why are Scrapy, BeautifulSoup, Selenium and other libraries identified as high-risk? How do bot developers deal with these restrictions?

12、[Elliptical Python Programming](https://susam.net/elliptical-python-programming.html)

`--(...==...)` equals 1, `--(...==...)--(...==...)` equals 2. Using this, you can write a long string to represent "hello, world", but this obscure code is a satire of over-pursuing tricks while ignoring readability.

## [🐿️ Projects & Resources](https://xiaobot.net/p/python_weekly)

1、[lmgameGamingAgent: Computer gaming agents that run on your PC and laptops.](https://github.com/lmgame-org/GamingAgent)

Aims to provide an easy-to-deploy Computer Use Agent (CUA) solution that can run on PCs and personal laptops.

![](https://img.pythoncat.top/2025-04-11-computer-game.png)

2、[memo: Manage your Apple Notes and Reminders](https://github.com/antoniorodr/memo)

A simple command-line tool developed in Python for managing Apple Notes and Apple Reminders, creating, searching, and organizing your notes from the terminal.

3、[Nugget: Unlock the fullest potential of your device](https://github.com/leminlimez/Nugget)

A tool for iOS 17.0 to 18.1.1 devices that unlocks hidden device features through vulnerabilities (enabling Dynamic Island, device spoofing, dynamic wallpapers and AOD, etc.), and disables unnecessary system services. (2.5K stars)

4、[no-code-architects-toolkit: Integrate common APIs into free APIs](https://github.com/stephengpope/no-code-architects-toolkit)

Supports audio transcription, multi-language translation, adding subtitles to videos, and can perform complex media processing, suitable for content creation. It also supports cross-cloud file management, including Google Drive, Amazon S3, Google Cloud Storage, and Dropbox.

5、[curl-impersonate: A special build of curl that can impersonate Chrome & Firefox](https://github.com/lwthiker/curl-impersonate)

A special curl build that can impersonate four major browsers: Chrome, Edge, Safari, and Firefox. Can perform exactly the same TLS and HTTP handshakes as real browsers. (5.1K stars)

6、[Wikipedia-API: Python wrapper for Wikipedia](https://github.com/martin-majlis/Wikipedia-API)

Convenient for retrieving Wikipedia information with Python, supports extracting text, sections, links, categories, translations, etc. from Wikipedia.

7、[adk-python: For developing, evaluating and deploying complex AI agents](https://github.com/google/adk-python)

Google's open-source Agent Development Kit (ADK) helps build efficient, intelligent AI agents. Features: code-first, multi-agent architecture, rich ecosystem, flexible orchestration, native streaming support, state/memory and component management. (3.2K stars)

8、[smartfunc: Turn docstrings into LLM-functions](https://github.com/koaning/smartfunc)

Directly converts docstrings into LLM functions through decorators, supports Jinja2 templates, supports async functions, uses Pydantic models to define response structures, ensuring consistent output formats.

9、[textcase: A feature-rich Python text case conversion library](https://github.com/zobweyt/textcase)

Supports conversion between multiple case formats (like snake_case, kebab-case, camelCase, etc.), intelligently recognizes abbreviations (like HTTPRequest) and supports non-ASCII characters.

10、[BabelDOC: PDF Document Translator](https://github.com/funstory-ai/BabelDOC)

PDF scientific paper translation and bilingual comparison library. Has online immersive translation and supports self-deployment. (1.6K stars)

![](https://img.pythoncat.top/2025-04-11-babeldoc.png)

11、[arborparser: Parse hierarchical structured text into tree representation](https://github.com/Jimmy228676/arborparser/tree/main)

Parses structured text and converts them into tree forms based on hierarchical headings, suitable for processing documents with nested headings like outlines, reports, and technical documentation. (Shared by @Enoch)

12、[DeepSeek-V3: DeepSeek Open Source Large Model](https://github.com/deepseek-ai/DeepSeek-V3)

Powerful Mixture of Experts (MoE) large model with 671B total parameters, using Multi-Head Latent Attention (MLA) and DeepSeekMoE architecture, pioneering auxiliary-loss-free load balancing strategy. (95.4K stars)
