## [🦄 Articles & Tutorials](https://xiaobot.net/p/python_weekly)

1、[How to run Python in production](https://ashishb.net/programming/python-in-production-2/)

The author shares practical experience running Python in production environments, covering multiple aspects including improving project quality, ensuring maintainability, and operational deployment.

2、[Browser Use Analysis - Why Could a Small Project Raise $17 Million?](https://blog.cnbang.net/tech/4241/)

Browser Use is a recent viral star project - a pure technical open-source project by two people with 8,000 lines of core code that raised $17 million! What exactly did it do, and why is it worth so much?

3、[Django Admin Theme Roundup 2025](https://www.djangoproject.com/weblog/2025/apr/18/admin-theme-roundup/)

Shares some Django Admin themes: Dracula, Daisy, django-jazzmin, django-admin-kubi, django-jet-reboot, django-semantic-admin, Simple UI, Grapelli, django-admin-interface, Unfold.

4、[15,000 lines of verified cryptography now in Python](https://jonathan.protzenko.fr/2025/04/18/python.html)

Introduces the journey of Python adopting the verified HACL cryptography library. The author proposed replacing Python's hash implementation in 2022 due to SHA3 vulnerabilities, and after 2.5 years of effort, Python's default hash and HMAC algorithms are now provided by HACL.

5、[Frankenstein's `__init__`](https://ohadravid.github.io/posts/2025-04-19-frank/)

Inspired by last week's article suggesting not to write `__init__` methods, this author shares an extreme `__init__` case they wrote, explaining why the code had errors.

6、[Sneak peek: A new ASN.1 API for Python](https://blog.trailofbits.com/2025/04/18/sneak-peek-a-new-asn.1-api-for-python/)

Python is about to introduce a new ASN.1 API that improves performance with a pure Rust parser, reduces discrepancy risks by sharing parsers with PyCA Cryptography, and enhances usability with a dataclass-style interface.

7、[Regex affordances](https://nedbatchelder.com/blog/202504/regex_affordances.html)

Shares a case study of using regular expressions for text manipulation, introducing several practical features like enabling verbose mode for better readability, capturing text through named groups, and optimizing matching logic with non-capturing groups.

8、[The next big thing in Python](https://www.bitecode.dev/p/the-next-big-thing-in-python)

The author reflects on the wonderful journey of Python's evolution over 30+ years while pondering how Python should develop and face challenges in the future. The author calls for simplifying program distribution and improving mobile platform development experience.

9、[Bootstrapping Python projects with copier](https://blog.dusktreader.dev/2025/04/06/bootstrapping-python-projects-with-copier/)

Introduces how to use the Copier tool to generate Python project templates with one click, integrating README, LICENSE, project structure, dynamic configuration, CI workflows, and other basic configurations for automated project scaffolding.

10、[ProcessThreadPoolExecutor: when I‍/‍O becomes CPU-bound](https://death.andgravity.com/ptpe)

Explores the problem of encountering CPU bottlenecks when handling I/O-intensive tasks in Python, and proposes a solution combining multithreading and multiprocessing: using thread pools in multiple processes to fully utilize CPU resources, solving the problem of high single-thread CPU usage.

11、[LLM Agents are simply Graph — Tutorial For Dummies](https://zacharyhuang.substack.com/p/llm-agent-internal-as-a-graph-tutorial)

How do AI agents work? The article explains in an accessible way, revealing that agents are actually directed graphs composed of nodes, branches, and loops, and demonstrates with code how to define nodes, build, and execute an agent.

12、[10,000-Word Deep Dive: Conversations with 50 Google Executives Reveal Google's "AI Catch-Up Journey"](https://mp.weixin.qq.com/s/-9Feb31tvo-598XTIAG1xw)

A story about Google's journey from the crisis triggered by ChatGPT's release to re-establishing AI leadership, featuring detailed recollections from 50+ employees/executives, showcasing Google's turbulent two-year period, the trade-offs and decisions made.
