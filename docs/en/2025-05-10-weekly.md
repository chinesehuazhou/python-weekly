## [🦄 Articles & Tutorials](https://xiaobot.net/p/python_weekly)

1、[Python 3.14 New Features Overview - What's Been Updated?](https://pythoncat.top/posts/2025-05-08-python-3.14)

Python 3.14 has entered feature freeze with many updates. Which ones deserve special attention? This article comprehensively reviews the changes in this version: lazy evaluation of type hints, brand new tail-call interpreter, experimental JIT compiler, non-intrusive debugging interface, REPL syntax highlighting, template strings, Zstandard compression, and more.

2、[Python π](https://iscinumpy.dev/post/python-314/)

Another overview of Python 3.14's main features. Compared to the previous share, it also introduces many smaller feature changes, making it good for comparative reading. Additionally, it's worth noting that the blogger has also published new feature overviews for all major versions since Python 3.7.

3、[Goodbye PyCon APAC: A New Chapter Begins](https://iqbalabdullah.net/en/posts/2025/03/the-final-pycon-apac/)

Reviews the 15-year development journey of PyCon APAC (PyCon Asia-Pacific Organization) since its founding in Singapore in 2010. The community has now been renamed to Python Asia Organization (PAO), and the article explains the reasons and related plans. (Additional: PyCon JP member commentary, [What Does PAO Mean for Japan's Python Community?](https://www.bilibili.com/video/BV1P6VzziEbE)) (Shared by @NoHeartPen)

4、[Semantic Unit Testing](https://www.alexmolas.com/2025/04/09/semantic-unit-testing.html)

An innovative testing approach: having large models analyze whether code conforms to documentation descriptions. The author's suite library (Semantic UnIt TEsting) can discover potential issues by analyzing function code and documentation comments without actually running the code.

5、[Asyncio Demystified: Rebuilding it From Scratch One Yield at a Time](https://dev.indooroutdoor.io/asyncio-demystified-rebuilding-it-from-scratch-one-yield-at-a-time)

Provides an in-depth yet accessible explanation of Python asynchronous programming's underlying principles, building a simplified version of Asyncio step by step. Through minimal code, it recreates Asyncio's core mechanisms, helping readers thoroughly understand the "magic" behind await.

6、[Python Project Management and Packaging: PEP 751 update](https://reinforcedknowledge.com/python-project-management-and-packaging-pep-751-update-and-some-of-the-remaining-issues-of-packaging/)

Introduces the latest developments in Python dependency management, particularly the PEP 751 standard lock file format. The article explains in detail the necessity of lock files, the characteristics of pylock.toml and the problems it solves, and explores some remaining challenges in Python packaging, such as native dependencies and accelerator dependency packages.

7、[Fiddling Weights with Temperature](https://www.boristhebrave.com/2025/05/07/fiddling-weights-with-temperature/)

Introduces a simple yet useful weight adjustment technique in AI: using temperature parameters to control random selection. Temperature of 1 maintains original weights, less than 1 favors high-weight items, and greater than 1 tends toward uniform distribution.

8、[Building News Agents with MCP, Q, and tmux](https://eugeneyan.com//writing/news-agents/)

The author developed a news summary agent based on MCP and Amazon Q CLI, supporting parallel processing and automatic summarization. Uses tmux to manage multiple processes, with each sub-agent responsible for a portion of news sources, and a main agent finally integrating to generate daily news.

9、[Scaling Python Task Queues Effectively](https://judoscale.com/blog/scaling-python-task-queues)

Explores Python task queue scalability issues, comparing the characteristics of Redis and RabbitMQ as message middleware. Also introduces two strategies: vertical scaling (increasing resources for individual workers) and horizontal scaling (increasing worker count), along with best practices for scaling Celery and RQ.

10、[What's new in pip 25.1 - Dependency groups!](https://ichard26.github.io/blog/2025/04/whats-new-in-pip-25.1/)

pip 25.1 brings several important updates: support for PEP 735 dependency group management; new package installation progress bars; support for resumable downloads; and support for generating PEP 751 standard lock files. Additionally, starting from this version, Python 3.8 is no longer supported.

11、[Quirks in Django's template language](https://blog.lilyf.org/posts/django-template-quirks/)

The author is rewriting Django's template engine in Rust and discovered several interesting edge cases: scientific notation doesn't support negative exponents, the upper filter is unsafe for HTML entities, and exception handling in if tags leads to counterintuitive results.

12、[Top Python Code Quality Tools](https://techbeamers.com/python-code-quality-tools/)

Introduces a Python code quality toolchain: static analysis (Pylint, Flake8, Bandit), code formatting (Black, isort), type checking (Mypy, Pytype), security scanning (Safety, Snyk), test coverage (Coverage.py), performance profiling (cProfile), etc.

## [🐿️ Projects & Resources](https://xiaobot.net/p/python_weekly)

1、[ty: An extremely fast Python type checker and language server, written in Rust.](https://github.com/astral-sh/ty)

From Astral team (creators of Ruff and uv), finally bringing the Rust wind to static type checking tools! Currently pre-release and still in active development. (4.2K stars)

2、[WeClone: Create Digital Avatars Using WeChat Chat Records](https://github.com/xming521/WeClone)

Uses WeChat chat records to fine-tune large language models, giving the model "that flavor," and binds it to a chatbot to create your own digital avatar. (3.3K stars)

3、[AI-Media2Doc: Convert Audio/Video to Xiaohongshu/WeChat Articles/Knowledge Notes/Mind Maps](https://github.com/hanshuaikang/AI-Media2Doc)

AI-based audio-video to graphic content creation assistant, with Vue-developed web frontend for local deployment without login registration. Supports multiple document styles including Xiaohongshu/WeChat articles/knowledge notes/mind maps/content summaries.

![AI-Generated Mind Map](https://img.pythoncat.top/2025-05-09-media2doc.png)

4、[blast: Browser-LLM Auto-Scaling Technology](https://github.com/stanford-mast/blast)

Adds AI web browsing capabilities to your applications, compatible with OpenAI API providing concurrent and streaming processing, with automatic caching and parallelization to reduce costs.

5、[contextgem: ContextGem: Effortless LLM extraction from documents](https://github.com/shcherbak-ai/contextgem)

Extracts structured data and insights from text and images with minimal code, supporting neural segmentation, concurrent IO processing, nested context extraction, multilingual support, etc.

6、[Fluent-M3U8: A cross-platform m3u8/mpd downloader based on PySide6 and QFluentWidgets.](https://github.com/zhiyiYo/Fluent-M3U8)

Based on PySide6 and QFluentWidgets development, with clean and user-friendly GUI, compatible with M3U8 and MPD videos, multi-threaded M3U8 downloading, download task management. (1K stars)

7、[Python-UIAutomation-for-Windows: Python 3 wrapper of Microsoft UIAutomation](https://github.com/yinkaisheng/Python-UIAutomation-for-Windows)

Wraps Microsoft's UIAutomation interface for automating various Windows desktop applications with Python scripts, including MFC, Windows Form, WPF, Modern UI (Metro UI), Qt, IE, Firefox, Chrome, and Electron-based applications. (2.9K stars)

8、[video-subtitle-extractor: Extract Hard-Coded Subtitles from Videos, Generate SRT Files](https://github.com/YaoFANGUK/video-subtitle-extractor)

Localized video hard-subtitle extraction tool that automatically detects subtitle regions and recognizes subtitle content without relying on third-party online OCR services. Supports watermark removal, logo removal, duplicate subtitle removal, and GPU acceleration. (7.2K stars)

9、[epub-utils: A Python library and CLI tool for inspecting ePub from the terminal](https://github.com/ernestofgonzalez/epub-utils)

Python-written EPUB processing tool for parsing and inspecting EPUB e-books from the terminal. Supports parsing and validating metadata like titles, authors, identifiers, and can highlight XML content.

10、[novelWriter: novelWriter is an open source plain text editor designed for writing novels.](https://github.com/vkbo/novelWriter)

Python-developed plain text editor that organizes and manages long novels by splitting them into multiple small documents, using Markdown-like minimal markup syntax with extended metadata annotations, synopsis, and cross-reference features. Supports multiple platforms (Linux, Windows, MacOS). (2.4K stars)

![](https://img.pythoncat.top/2025-05-09-novel-writer.png)

11、[asciimatics: Cross-platform Text UI & ASCII Animation Library](https://github.com/peterbrittain/asciimatics)

Develops full-screen text interfaces and ASCII animations, supporting colored text, cursor control, keyboard and mouse input, window adaptation, screen capture, plus rich animation effects, image-to-ASCII conversion, form controls, and other advanced features. Suitable for command-line tools, interactive forms, retro-style animations, etc. (4K stars)

An online PDF document "Prompt Engineering" from Google's prompt engineering white paper.
