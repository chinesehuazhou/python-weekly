# Python Trending Weekly #103: LlamaIndex's Monorepo Transformation

x
This week we're sharing 12 articles, 13 open-source projects, and 2 audio/video resources.

## [🦄Articles & Tutorials](https://weekly.pythoncat.top)

1、[Python Tooling at Scale: LlamaIndex's Monorepo Overhaul](https://www.llamaindex.ai/blog/python-tooling-at-scale-llamaindex-s-monorepo-overhaul)

To tackle dependency tree issues across 650+ Python packages, the LlamaIndex team developed LlamaDev as a Pants replacement and introduced uv to replace Poetry, solving problems with slow build times, complex dependency management, and unstable CI.

2、[Recent disruptive changes from Setuptools](https://lwn.net/SubscriberLink/1020576/fcef31015579a9b1/)

Early Setuptools versions treated hyphens and underscores in setup.cfg configuration file keys as equivalent, later adding deprecation warnings for hyphens. The recent 78.0.1 release turned deprecation into errors, affecting 12,000 packages...

3、[New tools and features in the Responses API](https://openai.com/index/new-tools-and-features-in-the-responses-api/)

OpenAI has updated the Response API with new features including remote MCP server support, image generation, code interpreter, plus new capabilities like background mode, reasoning summaries, and encrypted reasoning.

4、[Python: a quick cProfile recipe with pstats](https://adamj.eu/tech/2025/05/20/python-quick-cprofile-recipe-pstats/)

A guide on efficiently analyzing Python script performance using cProfile and pstats command line, supporting various sorting methods and result saving. Demonstrates how to identify slow function calls through a Django database migration example.

5、[PEPs & Co](https://hugovk.dev/blog/2025/peps-and-co/)

Explores the origins of Python Enhancement Proposals (PEPs) - their background and naming rationale - and their influence on numerous open-source communities, listing similar proposal mechanisms across various open-source projects.

6、[How we make decisions in Django](https://buttondown.com/carlton/archive/how-we-make-decisions-in-django/)

A summary from the author's DjangoCon Europe 2025 talk, highlighting issues with Django's complex contribution process, lengthy feature discussion cycles, and strict code reviews. Suggests strengthening third-party ecosystems, reconsidering the "batteries included" philosophy, and exploring modular development.

7、[PEP 791 – imath — module for integer-specific mathematics functions](https://peps.python.org/pep-0791/)

This PEP proposes introducing a new imath module in Python 3.15, specifically handling integer-related mathematical functions like combinations, factorials, and greatest common divisors, simplifying the math module's complexity and centralizing related functions.

8、[Loading Pydantic models from JSON without running out of memory](https://pythonspeed.com/articles/pydantic-json-memory/)

Loading a 100MB JSON file with Pydantic typically consumes up to 2000MB of memory! The author introduces two optimization strategies: using ijson for streaming parsing and dataclasses with slots. Combined, these can reduce memory usage to 450MB.

9、[From script to version 1.0.0 — 12 years later](https://wirtel.be/post/2025/05/20/2025-dsmtpd-version-1/)

The author open-sourced a Python SMTP debugging tool in 2013, then suddenly received an amazing PR this year, prompting the project's 1.0.0 release. Even tiny open-source projects can have lasting impact - isn't that the magic of open source?

10、[Efficiently Mirror the Entire PyPI Repository with Bash](https://www.repoflow.io/blog/efficiently-mirror-the-entire-pypi-repository-with-bash)

Shows how to efficiently mirror the entire PyPI repository to a local directory using a Bash script, meeting Python package management needs in offline or high-security environments.

11、[A simple search engine from scratch](https://bernsteinbear.com/blog/simple-search)

How do you build a simple search engine with Python? The author shares a detailed approach using Word2Vec embedding technology, covering vectorization, similarity calculations, and other technical details.

12、[Working on Complex Systems: What I Learned Working at Google](https://www.thecoder.cafe/p/complex-systems)

The author shares insights on complex systems, summarizing five key characteristics (emergent behavior, delayed consequences, local vs. global optimization conflicts, lag effects, fuzzy boundaries) and several coping strategies (embracing uncertainty, building feedback mechanisms, encouraging collaboration and knowledge sharing).

## [🐿️Projects & Resources](https://weekly.pythoncat.top)

1、[Index-anisora: The Ultimate Open-Source Anime Video Generation Model](https://github.com/bilibili/Index-anisora)

Bilibili's anime generation model - currently the most powerful open-source animation generation model. Generate various anime-style video clips with one click, including anime episodes, Chinese animation, manga adaptations, VTuber content, anime PVs, and MAD videos.

2、[DeepResearchAgent: DeepResearchAgent](https://github.com/SkyworkAI/DeepResearchAgent)

A hierarchical multi-agent system designed for deep research tasks and general-purpose tasks. The framework uses top-level planning agents to coordinate multiple specialized bottom-level agents, enabling automated task decomposition and efficient execution across complex domains.

3、[workflow-use: Create and run workflows (RPA 2.0)](https://github.com/browser-use/workflow-use)

From the Browser Use team - the simplest way to create and execute deterministic workflows, with fallback to Browser Use if steps fail. Create automated workflows based on a single screen recording. (2.7K stars)

4、[tchMaterial-parser: SmartEdu Textbook PDF Downloader](https://github.com/happycola233/tchMaterial-parser)

A download tool for electronic textbooks from China's National Smart Education Platform. Supports batch downloads, automatic naming, cross-platform compatibility, and high-resolution adaptation. Simple operation with login support and visual download progress. (2.1K stars)

5、[mistletoe: A fast, extensible and spec-compliant Markdown parser in pure Python.](https://github.com/miyuchina/mistletoe)

A CommonMark-compliant Markdown parser implemented in pure Python. Fast, extensible, supporting multiple output formats and custom rendering - perfect for scenarios requiring Markdown conversion to HTML, LaTeX, and other formats.

6、[PurpleLlama: Set of tools to assess and improve LLM security.](https://github.com/meta-llama/PurpleLlama)

Meta's PurpleLlama project aggregates various tools for enhancing large model security, including input/output content security detection, code security protection, malicious prompt injection defense, etc. Supports multiple languages and large context windows - ideal for developers and teams building security defenses for LLM applications. (2.8K stars)

7、[PPTAgent: Generating and Evaluating Presentations Beyond Text-to-Slides](https://github.com/icip-cas/PPTAgent)

Supports generating structured, richly illustrated presentations from documents, with built-in multi-dimensional quality assessment tools, reference learning, and intelligent content analysis. (1.4K stars)

![PPT generation workflow by agent](https://img.pythoncat.top/2025-05-23-ppt_agent.jpg)

8、[buckaroo: The data table UI for Notebooks](https://github.com/paddymul/buckaroo)

Brings high-performance, infinitely scrollable data tables to Jupyter and other data analysis environments. Supports mainstream DataFrames like pandas and polars, with built-in sorting, searching, statistics, histograms, auto-cleaning, and low-code UI.

9、[agent-squad: Flexible and powerful framework for managing multiple AI agents and handling complex conversations](https://github.com/awslabs/agent-squad)

For coordinating multiple AI agents to handle complex conversations, intelligently routing queries and maintaining context across different interactions. (5.8K stars)

10、[juvio: UV kernel for Jupyter](https://github.com/OKUA1/juvio)

A specially designed kernel for reusable, dependency-aware, and Git-friendly Jupyter Notebooks. Supports inline dependency management, automatic environment setup, and converts notebooks to script format for easy version control.

11、[NLWeb: Natural Language Web](https://github.com/microsoft/NLWeb)

Dedicated to enabling websites to provide natural language conversation interfaces. Based on open protocols and schema.org semantic layers, supports natural interaction between humans and AI agents via MCP protocol, enabling websites to quickly achieve semantic access in the AI Web era. (2.8K stars)

12、[pyftpdlib: Extremely fast and scalable Python FTP server library](https://github.com/giampaolo/pyftpdlib/)

Provides high-level, portable interfaces for easily writing asynchronous FTP servers. Complete RFC-959 server implementation supporting FTPS, IPv6, Unicode filenames, and more. (1.7K stars)

13、[The Way of Code](https://www.thewayofcode.com/)

The Claude team just released "The Way of Code" website, content inspired by the Tao Te Ching, incorporating philosophical reflections on programming. Each chapter features dynamic visuals that match the mood - incredibly creative.
