## [🦄Articles & Tutorials](https://weekly.pythoncat.top)

1、[Design Patterns You Should Unlearn in Python-Part1](https://www.lihil.cc/blog/design-patterns-you-should-unlearn-in-python-part1/)

Many Java/C++ design patterns just don't vibe with Python's philosophy. The author compares traditional implementations of Singleton and Builder patterns with their Pythonic alternatives, showing how to leverage Python's module system, closures, and default parameters to write cleaner, more elegant code while dodging unnecessary complexity and potential pitfalls.

2、[The Python Language Summit 2025](https://pyfound.blogspot.com/2025/06/python-language-summit-2025.html)

The annual gathering of Python core developers, diving deep into the game-changing free-threading revolution. The summit covered cutting-edge topics like concurrency race handling, fearless concurrency, and governance challenges, with special focus on multi-threading performance optimization and language evolution strategies.

3、[State of Free-Threaded Python](https://pyfound.blogspot.com/2025/06/python-language-summit-2025-state-of-free-threaded-python.html)

Core developer Matt Page presented the latest updates on free-threaded Python at the Language Summit, evaluating PEP 703's implementation across four key dimensions: performance, stability, maintainability, and adoption. Single-threaded performance overhead has dropped from 40% to under 10%, and the architectural design has proven its worth.

4、[An introduction to Python for R users](https://occasionaldivergences.com/posts/python-intro/)

A practical guide written by an author who switched from teaching R to Python, helping R users get up to speed quickly by contrasting the design philosophies and syntax differences between the two languages. It covers environment setup, data processing library choices, and the mental shift from functional to object-oriented programming.

5、[How global variables work in Python bytecode](https://fromscratchcode.com/blog/how-global-variables-work-in-python-bytecode)

A deep dive into how the Python virtual machine handles global variables, revealing the fundamental differences between global and local variables through bytecode analysis. The VM performs dynamic name resolution for globals rather than index-based access, and this extra layer of indirection is what makes Python's dynamic features possible.

6、[Are Python Dictionaries Ordered Data Structures?](https://www.thepythoncodingstack.com/p/are-python-dictionaries-ordered-data)

Before Python 3.6, dictionaries were unordered. Starting with 3.6, they began preserving insertion order, but only as an implementation detail. From 3.7 onwards, insertion order became part of the official language spec. However, dictionary ordering doesn't affect equality comparisons—unlike sequences such as lists, dictionaries are fundamentally about key-value mapping.

7、[Understanding and Coding the KV Cache in LLMs from Scratch](https://magazine.sebastianraschka.com/p/coding-the-kv-cache-in-llms)

KV cache dramatically speeds up LLM inference by storing and reusing previously computed key-value vectors, eliminating redundant calculations. The article breaks down the redundant computation problem in attention mechanisms and provides complete Python code for implementing KV cache from scratch.

8、[从 browser-use 出发，品 Agent 实现](https://www.bestblogs.dev/article/7d6305)

Using the open-source browser-use project as a case study, this article dives deep into LLM Agent implementation principles and engineering practices. It dissects browser-use's architecture, including Agent Core, MessageManager, Memory, LLM Interface, Controller, and BrowserContext components and their interaction flows.

![browser-use](https://img.pythoncat.top/2025-06-21-browser-use.avif)

9、[PEP 795 – Deep Immutability in Python](https://pep-previews--4468.org.readthedocs.build/pep-0795/)

A PEP draft proposing deep immutability mechanisms through a freeze() function that recursively freezes objects and all their referenced objects. This proposal tackles data integrity protection and thread-safe sharing issues, providing robust support for functional programming and concurrency optimization.

10、[The Missing Manual for Signals: State Management for Python Developers](https://bui.app/the-missing-manual-for-signals-state-management-for-python-developers)

An introduction to using Signals for state management in Python, which solves dependency coordination issues in traditional state management through reactive programming. When state changes, it automatically triggers related updates without manual maintenance. This reactive mechanism prevents the common "forgot to update X when Y changed" problem.

11、[Create your customized running plan: A step-by-step guide using Python, Elasticsearch, and Agno](https://allthingsopen.org/articles/step-by-step-guide-python-elasticsearch-agno-agentic-ai-create-running-plan)

Learn how to build an AI running coach by parsing Apple Health XML data into Elasticsearch, then using the Agno framework to generate personalized four-week training plans. Finally, convert the Markdown-formatted plans into Notion pages for a complete data-driven fitness solution.

12、[The fastest way to detect a vowel in a string](https://austinhenley.com/blog/vowels.html)

The article explores 11 different methods for detecting vowels in strings, with benchmarking and bytecode analysis revealing that regex performs best on long strings. It exposes how Python interpreter overhead significantly impacts performance and showcases CPython's regex engine optimization tricks using bitmap lookups.
